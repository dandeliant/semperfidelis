# Semper Fidelis 🕊️

**Przewodnik Chrześcijanina** — piękna, interaktywna aplikacja internetowa (PWA) ze zbiorem modlitw, pieśni, wiedzy o wierze i ciekawostek w pięciu językach: 🇵🇱 polskim, 🇬🇧 angielskim, 🇷🇺 rosyjskim, 🇩🇪 niemieckim i 🇫🇷 francuskim.

**➡ Aplikacja na żywo: [dandeliant.github.io/semperfidelis](https://dandeliant.github.io/semperfidelis/)**

## Funkcje

- 📿 **Modlitwy** — Ojcze nasz, Zdrowaś Maryjo, Aniele Boży, modlitwa do św. Michała Archanioła i inne
- 🎵 **Pieśni** — tradycyjne pieśni religijne, inne dla każdego języka
- 📖 **Wiedza o wierze** — rok liturgiczny, sakramenty, aniołowie
- ✨ **Ciekawostki** — fakty z dwóch tysięcy lat historii chrześcijaństwa
- 🕯 **Kaplica światła** — zapal wirtualną świecę w swojej intencji
- 🌍 **5 języków** z pełnym tłumaczeniem treści
- 📱 **PWA** — zainstaluj na telefonie i korzystaj offline

## Instalacja na telefonie

1. Otwórz [stronę](https://dandeliant.github.io/semperfidelis/) w przeglądarce (Chrome / Safari).
2. **Android (Chrome):** menu ⋮ → „Dodaj do ekranu głównego" / „Zainstaluj aplikację".
3. **iPhone (Safari):** przycisk Udostępnij → „Dodaj do ekranu początkowego".

## Technologia

Czysty HTML + CSS + JavaScript, bez frameworków. Service worker zapewnia działanie offline, a manifest — instalację jako aplikacja.
